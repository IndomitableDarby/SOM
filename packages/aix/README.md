Som RPM packages for IBM AIX
==============================

In this repository, you can find the necessary tools to build a Som package for AIX.

## Building AIX packages

Please, visit the following link for the full AIX packages building documentation: [Generate Som AIX packages.](https://documentation.som.com/current/development/packaging/generate-aix-package.html)
